Spordle

This is a project that's constantly in progress. View website here

Spordle is a web app using ASP.NET Core, GraphQL, and Razor Pages. It is hosted on Azure Cloud Services and uses Spotify's API to interact with the users. I thought of this project during the craze over the viral game wordle. The idea is that you get to build a playlist using the website that is progressively built upon everyday given a couple of songs from different genres. The songs and their uri's are stored on a Azure SQL Database and everyday the songs change. (At the current moment, the songs only change every day of the week.) I plan to continue working on this project in order to further advance my skills in Web Design in the Front and Backend.
